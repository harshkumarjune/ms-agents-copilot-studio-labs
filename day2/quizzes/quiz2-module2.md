### Q1. Copilot Studio Evolution

What was Microsoft Copilot Studio previously known as?

- A) Microsoft Bot Framework
- B) Power Virtual Agents
- C) Azure Bot Service
- D) Dynamics 365 Virtual Agent

**Answer:** B

### Q2. Pro-Code vs Low-Code

When should you use pro-code capabilities in Copilot Studio? (Select all that apply)

- A) When you need custom API integrations
- B) When the visual designer is sufficient
- C) When you need advanced orchestration logic
- D) When integrating custom AI models
- E) When building a simple FAQ bot

### Q3. Model Context Protocol (Definition)

- A) A Microsoft proprietary encryption protocol
- B) A standardized protocol for connecting AI agents to external tools and data sources
