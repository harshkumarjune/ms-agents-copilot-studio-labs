# Microsoft 365 Agents Pro-Code Training

Welcome to the Microsoft 365 Agents Pro-Code Training! This repository contains all the labs, code samples, and training materials you'll need for the course.

## Course Overview

This 2-day intensive training program covers building intelligent agents for Microsoft 365 Copilot using pro-code approaches.

### Day 1: Foundations & Declarative Agents
- Understanding Copilots vs Agents
- Microsoft 365 Copilot Extensibility
- M365 Agents Toolkit
- TypeSpec for API definitions
- Semantic Kernel basics

**Labs:**
- Lab 1: Build a Repair Service Declarative Agent
- Lab 2: Create a Geo Locator Game Agent
- Lab 3: Custom Engine Agent with Semantic Kernel

### Day 2: Advanced Topics & Copilot Studio
- Model Context Protocol (MCP)
- Azure AI Search & RAG patterns
- Copilot Studio pro-code features
- Bring Your Own Model (BYOM)

**Labs:**
- Lab 4: Deploy MCP Server for Copilot Studio
- Lab 5: Retail Agent with Azure AI Search & BYOM

## Prerequisites

- Node.js 18+ and npm
- Visual Studio Code
- M365 Agents Toolkit extension
- Azure subscription (for labs 4-5)
- Microsoft 365 developer tenant

## Getting Started

1. Clone this repository
2. Open in VS Code
3. Follow the lab guides in order (Lab 1 → Lab 5)

Open the HTML slide files in a browser for the presentation slides. Use arrow keys to navigate.
